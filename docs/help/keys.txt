Mouse in Langband (only SDL):
=============================

left mouse button  - will display info on the clicked object/symbol.
right mouse button - will try to target creature or location


Normal Keyset in Langband:
==========================

a - aim/zap a wand/staff/rod
b - browse spells
c - close door
d - drop item
e - show equipment
f - fire missile/missile-weapon
g - get item on the ground
i - show inventory
k - crush/destroy item
m - invoke magic spell or prayer
o - open door
p - invoke magic spell or prayer
q - quaff potion
r - read text/scroll/leaflet
s - search area
t - take off a worn item
u - generic 'use item' command
v - throw item
w - wear item
z - aim/zap a wand/staff/rod

B - bash/smash down door
C - show character
D - disarm a trap/mechanism
E - eat food/corpse/curry
F - refill item (e.g torch or lamp)
L - learn a spell
Q - quit game (kills character and savegeame, use ^X for safe exit)
R - rest
S - save game

? - show this help
> - go downstairs
< - go upstairs
* - select target [also right mouse button]
, - stand still
. - toggle run-mode (also shift-arrowkey)

^A - debug commands [see wizard.lisp]
^P - show previous messages
^R - redraw screen
^T - toggle ascii/graphics map [SDL only]
^X - save and exit game
