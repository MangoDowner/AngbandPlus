Experimental binary for Win32

Brand new for Langband 0.1.6 is the option of an easy-to-install package for Windows. It requires that you have a Lisp-system installed, but that's easy to install too. For 0.1.6 the Win32 version does not have sound. This will be supported in the next version.

  • Lispworks Personal - which is simple to install and will run Langband just fine. Just grab the binary, you don't need the docs.
  • Langband binary install - A simple-to-use installer.

Debian-packages sid/sarge/unstable/testing

For users of Debian sid/sarge, langband is in the main repositories and can be downloaded like all other debian applications. Recommended lisp-systems to use langband on Debian sid are cmucl-normal and sbcl.

Debian-packages woody/stable

To get langband for debian woody you need to add a few lines to your /etc/apt/sources.list to get langband and to get the necessary lisp-systems. These packages are not official like the ones for Debian unstable. 

  deb http://www.langband.org/debian local lisp
  deb ftp://debian.b9.com stable main
  deb-src http://www.langband.org/debian local lisp
  deb-src ftp://debian.b9.com stable main
