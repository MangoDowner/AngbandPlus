# Portralis Stores Definitions.

# Each towns can have their own shops arrays.
# Array 0 is the default one in case the town doesn't have one.
# Due to the way this file is handled, array 0 should be
# defined at the end.

# There can be up to 100 items defined per store types. Define a single
# more often for it to have more chances to appear, and in larger quantities.

# By convention, the stores are the following:

# 0. General Store
# 1. Armory
# 2. Weaponry
# 3. Temple
# 4. Alchemy Shop
# 5. Magic Shop
# 6. Black Market(do not define any items for it)
# 7. Home(do not define any items for it)
# 8. Dungeon Shop(do not define any items for it)
# 9. Licialhyd Shop(hard-coded for the most part, don't edit)
# 10. Rare Items Shop
# 11. Rods and Crystals Shop
# 12. Music Shop

# T:(town's min):(town max)  -- This means that towns no. (min) to (max) will use this list.
# I:(shop's id):(item's id)
