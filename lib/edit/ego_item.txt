# File: ego_item.txt
# DaJAngband 1.2.2


# This file is used to initialize the "lib/data/ego_item.raw" file, which is
# used to initialize the "ego-item" information for the Angband game.

# Do not modify this file unless you know exactly what you are doing,
# unless you wish to risk possible system crashes and broken savefiles.
# After modifying this file, delete the "lib/data/ego_item.raw" file.

# The ego-item indexes are defined in "defines.h", and must not be changed
# or savefile compatibility will be lost.

# Note that several "ego-item" types may share the same "textual name",
# such as with "Armor of Resistance" and "Shield of Resistance".

# === Understanding ego_item.txt ===

# N: serial number : ego type
# R: randsus : randres : randlowr : randpow : randslay : randbran : randbon : randplu : randdrb
# X: esprace : randact : unused
# C: max to-hit : max to-dam : max to-ac : pval
# W: depth : rarity : rating : weight : cost
# T: tval : min_sval : max_sval
# F: flag | flag | etc

# 'N' indicates the beginning of an entry. The serial number must
# increase for each new item.

# *** NEW
# 'R' is for random attributes:
#   randsus = how many random sustains on the item (0, 1, or 2)
# Sustains prevent stat drains for strength, dexterity, constitution, wisdom,
# intelligence, or charisma.
#   randres = how many random high resists on the item (0, 1, 2, or 3)
# Random 'high' resistances include resistances to: poison, fear, light, dark,
# blindness, confusion, sound, shards, nexus, nether, chaos, disenchantment,
# charm, static (drain charges), slime, and silver poison.
#   randlowr = how many low random resists on the item (0, 1, or 2)
# 'Low' resistances are fire, cold, acid, and lightning.
#   randpow = how many random powers on the item (0, 1, or 2)
# Random powers include: slow digestion, feather falling, free action,
# see invisible, permanent light, regeneration, telepathy, hold life, darkvision,
# breath shield (damage reduction vs monster breath), teleport control and throwing might.
#   randslay = how many random slays on the item (0, 1, or 2)
# Random slays include slays against: orcs, trolls, giants, undead, demons, dragons,
# creatures vulnerable to silver, bugs, silver (grepse) monsters, and creatures of light.
#   randbran = whether there is a random brand on the item (always 0 or 1).
# Chooses a brand type from poison, fire, cold, acid, and lightning.
#   randbon = how many random bonuses on the item (0, 1, or 2)
# Random bonuses include bonuses to: magic device skill, stealth, alertness,
# luck, tunneling, and speed. These are modified by the item's pval.
#   randplu = how many random pluses on the item (0, 1, or 2)
# Random plusses are plusses to stats (strength, dexterity, constitution, wisdom,
# intelligence, or charisma) modified by the item's pval.
#   randdrb = how many random drawbacks on the item (0, 1, or 2)
# Random drawbacks include: corruption, aggravation, accidents (causes you to hit
# yourself sometimes), earthquake blows, prevent hp regeneration, random teleportation,
# experience drain, the enchantment of peace (reduces blows per turn by 1, and
# hinders combat in various other ways but has minor advantages also), or 
# random annoyances (once in awhile does something annoying).
#   You may have up to 3 random resistances on an ego, and up to 2 of any
# other random attribute.
#   Special: Random Immunity: If you put 3 for randdrb, the item will get 2 random
# drawbacks AND a random immunity. An ego may only have a random immunity if it also
# has the drawbacks.  randdrb has another magic number coded: If you put 11 for randdrb,
# then the item will have a random drawback about half the time.
#   Special: magic number 9: If you put a 9 for the number of any random attribute,
# (except randbran) Then it will have 1d2 random attributes of that type (or 1d3 for
# the random high resists).  It will read any other value you enter as 1, except
# randres and randdrb will read weird values as 2.
#   Any ego which has two random slays (or any other random attribute type) may end
# up getting only one because it drew the same random attribute twice.
#   *If an item gets regeneration as a random power and regeneration prevention as a
# random drawback, it will not remove either one because they both still have an effect:
# While hit point regeneration is stopped, regeneration will still made you regain mana
# faster.

# 'X' is for extra item powers: 
#   esprace is a code for items which have race-specific ESP, like this:
# 0-none, 1=orcs, 2=trolls, 3=giants, 4=undead, 5=dragons, 6=demons
# 7=light fairies, 8=bugs, 9=grepse (silver monsters),..
# Special esprace codes are: 40 = random esprace,
# 34 = 35% chance of ESP for undead, 
# 35 = 35% chance of ESP for dragons, 36 = 50% chance of ESP for demons
# 41 = matches one of the random slays, 
# 42 = 35% chance of having racial ESP matching one of the random slays.
# The range for race-specific ESP is only half the range of full ESP.
#   randact is not implemented yet and currently has no effect.  Maybe at some
# point I will add random activations for egos.
#   ...

# *** OLD
# X: rating : xtra
# 'X' is for extra information -  The xtra value determines random extra
# flags for the ego-item.  Nothing happens with a value of 0, 1 stands
# for "one random extra sustain", 2 is "one random extra resist", and 3 is
# "one random extra ability".

# 'C' is for creation bonusses - maximum to-hit, to-dam, to-ac, and
# pval. Cursed ego-items will negate the given value.
#  *Make sure staffs (tval 55) cannot get any ego that has a pval
# because pval is also used for staff charges.

# 'W' is for extra information. Depth is the depth the object is
# normally found at, rarity determines how common the object is,
# rating affects level feelings, weight is in tenth-pounds and cost is
# the items value.

# 'F' is for flags. These are fairly self-explanatory. As many F:
# lines may be used as are needed to specify all the flags and flags
# are separated by the '|' symbol.

# 'T' is for possible 'tval' and 'sval' values of the base-item.
# Up to five 'T' entries are possible for a single ego type. The
# ego-item can only be generated from items of this tval, with sval
# in the specified range.

#  The following ego weapon types have greater chance to get extra dice:
#(percent chance for automatic +1 dice is in addition to usual small chance of extra dice)
# 96  CRITICAL_WEIGHT (35%)
# 126 CRITICAL_AMMO (20%) (extra dice chance for throwing weapons, not missile ammo)
# 141 GORING (9%)
# 155 ASSASSIN (16%)
# 134, 135, 136, 137, 138, 147 all "of randomness" weapon egos (pseudo-randarts) (10%)
# (coded by ego index)
