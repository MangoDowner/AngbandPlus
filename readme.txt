To build FAangband under linux (or, I beleive, unix):

1. unzip and cd to top level directory
2. ./configure
3. make install

If this fails, try:

1. unzip and cd to top level directory
2. autoheader
3. aclocal
4. autoconf
5. automake -a
6. ./configure
7. make install
